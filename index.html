PagueVeloz PHP

Cliente PHP da API do PagueVeloz

View on GitHub Download .zip Download .tar.gz

Instalação

Instale usando o composer: composer require pagueveloz/pagueveloz

Comece a usar

//importe o autoload gerado pelo composer
require_once __DIR__."/vendor/autoload.php";

//importe a classe do PagueVeloz
use PagueVeloz\PagueVeloz;

//configure a classe para usar a url da API do PagueVeloz
PagueVeloz::Url('https://www.pagueveloz.com.br/api');
